Request a change to GOV.UK

Make a request to change content on GOV.UK if you've:

  • seen something that's factually wrong
  • evidence that the user experience isn't satisfactory, eg direct user feedback
  • found a genuine user need for content that isn't already on GOV.UK and the user need is in the proposition

Don't make a GOV.UK content request if you:

  • don't agree with the way something is written
  • think content should be moved on GOV.UK
  • are repeating content that can be found elsewhere on the internet
  • want to put up information that's only for people working in DWP
  • are doing so for business reasons at the expense of the user experience
  • want to support a publicity campaign

How to make a request

If you think your request meets the above criteria email Keith Prust (keith.prust@dwp.gsi.gov.uk) or, if the content is to do with a digital service, Tom Adams (tomadams.digitalDWP@gmail.com).

Explain the user need you've identified and give any evidence or data you have that users are looking for or need the change.

Don't include suggestions for how content should be written or 'mock-ups' of pages. These won't be acknowledged by GDS and so are a waste of time to produce.
