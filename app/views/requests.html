Request a change to GOV.UK

Make a request to change content on GOV.UK if you've:

  • seen something that's factually wrong
  • evidence that the user experience isn't satisfactory, eg direct user feedback
  • found a genuine user need for content that isn't already on GOV.UK and the user need should be met on GOV.UK

Don't make a GOV.UK content request if you:

  • don't agree with the way something is written
  • think content should be moved on GOV.UK
  • are repeating content that can be found elsewhere on the internet
  • want to put up information that's only for people working in DWP
  • are doing it for purely business reasons, eg removing a phone number to stop calls and not giving an alternative

How to make a request

If you think your request is valid email Keith Prust (keith.prust@dwp.gsi.gov.uk) or, if the content is to do with a digital service, Tom Adams (tomadams.digitalDWP@gmail.com).

Explain the need you've found and give any evidence or data you have that people need the change to be made.

Put any request for changes in as soon as possible. Allow 28 days for small changes to existing content, eg adding a sentence to an existing page, and 6 to 8 weeks for more complicated requests involving creating brand new pages on GOV.UK.

Don't include suggestions for how content should be written or 'signed off' content. A content designer will be given your request and will decide what content is needed.

Campaign content

Speak to Gillian Hudson, Strategic Communications Directorate Head of Digital, about options for digital campaign content, including:

  • use of social media
  • getting a campaign page on GOV.UK
